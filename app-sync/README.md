# AppSync

AppSync fornece uma interface GraphQL robusta e escalável para
desenvolvedores de aplicativos combinarem dados de diferentes
fontes, incluindo DynamoDB, Lambda e APIs HTTP.

## Features

- Edição avançada de schema GraphQL por meio do console AppSync

- Geração automática de schema GraphQL a partir do DynamoDB

- Suporte a escrever lógica de negócios (JavaScript/Typescript) que 
  acessa suas fontes de dados

- Cache eficiente

- Integração com grupo de usuários do Cognito para controle de acesso
  a nível de campo
